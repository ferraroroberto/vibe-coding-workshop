## English

# Exercise: The Great Merger

## The Goal
In this exercise, you will automate a common reporting task: consolidating multiple Excel files into a single dataset. You will learn how to load data from Excel, combine it using Python, and export the cleanup result. 

## The Scenario
Your boss has emailed you three separate Excel files containing quarterly sales data (Q1, Q2, and Q3). She needs a single "Year-To-Date" report by lunchtime. 

Copy-pasting manually is slow and prone to errors. Instead, you will use Python to:
1. Load the three files.
2. Stack them into one master table.
3. Verify the data count.
4. Export the result to a CSV file.

## Expected Output
A single file named `combined_sales.csv` containing all records from the three quarterly files.

## Español

# Ejercicio: La Gran Fusión

## El Objetivo
En este ejercicio automatizarás una tarea común de reportes: consolidar múltiples archivos Excel en un único conjunto de datos. Aprenderás a cargar datos desde Excel, combinarlos con Python y exportar el resultado limpio.

## El Escenario
Tu jefa te ha enviado por correo tres archivos Excel separados con datos de ventas trimestrales (Q1, Q2 y Q3). Necesita un único reporte "Acumulado del Año" antes del almuerzo.

Copiar y pegar manualmente es lento y propenso a errores. En su lugar, usarás Python para:
1. Cargar los tres archivos.
2. Apilarlos en una tabla maestra.
3. Verificar el conteo de datos.
4. Exportar el resultado a un archivo CSV.

## Resultado Esperado
Un único archivo llamado `combined_sales.csv` que contenga todos los registros de los tres archivos trimestrales.
