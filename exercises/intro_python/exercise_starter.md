## English

# Intro: Hello World - The First Flight

## The Goal

Verify that your "cockpit" (VS Code + Python) is fully operational. You will write your first lines of code to print a message to the screen, perform a business calculation, and automate a repetitive task using a loop.

## Steps

1. **The Environment:** Create a new file named `hello.py`.
2. **The First Prompt:** Use GitHub Copilot to ask: "How do I print 'Hello World' in Python?". Run the code.
3. **The Calculator:** Ask Copilot: "Create a variable for revenue = 1000 and cost = 400. Calculate profit and print it."
4. **The Loop:** Ask Copilot: "Write a loop that prints numbers 1 to 5."

## Español

# Intro: Hola Mundo - El Primer Vuelo

## El Objetivo

Verifica que tu "cabina" (VS Code + Python) esté completamente operativa. Escribirás tus primeras líneas de código para imprimir un mensaje en pantalla, realizar un cálculo de negocio y automatizar una tarea repetitiva usando un bucle.

## Pasos

1. **El Entorno:** Crea un nuevo archivo llamado `hello.py`.
2. **El Primer Prompt:** Usa GitHub Copilot para preguntar: "¿Cómo imprimo 'Hello World' en Python?". Ejecuta el código.
3. **La Calculadora:** Pide a Copilot: "Crea una variable para ingresos = 1000 y costos = 400. Calcula el beneficio e imprímelo."
4. **El Bucle:** Pide a Copilot: "Escribe un bucle que imprima los números del 1 al 5."
