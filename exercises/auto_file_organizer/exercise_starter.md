## English

# Exercise: The automated File Organizer

## The Goal

Your "Downloads" folder and project workspace have become a chaotic mess of files. 
There are CSVs mixed with PDFs, Excel reports, and random text files. It's time to clean up.

**Your Objective:** Write a Python script that automatically organizes these files into neat subfolders based on their type (e.g., all `.csv` files go into a `Data` folder, `.pdf` into `Docs`).

### Expected Outcome

When you run your script, the files in the `data` folder should be moved into these new subfolders:
*   **Spreadsheets/** (for `.xlsx`, `.csv`)
*   **Documents/** (for `.pdf`, `.txt`, `.docx`)
*   **Images/** (for `.png`, `.jpg`)
*   **Others/** (for anything else)

Good luck!

## Español

# Ejercicio: El Organizador Automático de Archivos

## El Objetivo

Tu carpeta "Descargas" y el espacio de trabajo del proyecto se han convertido en un caos de archivos.
Hay CSVs mezclados con PDFs, reportes de Excel y archivos de texto aleatorios. Es hora de ordenar.

**Tu Objetivo:** Escribir un script en Python que organice automáticamente estos archivos en subcarpetas ordenadas según su tipo (ej.: todos los `.csv` van a una carpeta `Data`, los `.pdf` a `Docs`).

### Resultado Esperado

Cuando ejecutes tu script, los archivos en la carpeta `data` deben moverse a estas nuevas subcarpetas:
*   **Spreadsheets/** (para `.xlsx`, `.csv`)
*   **Documents/** (para `.pdf`, `.txt`, `.docx`)
*   **Images/** (para `.png`, `.jpg`)
*   **Others/** (para todo lo demás)

¡Buena suerte!
